## 6. 请求处理流程

```
用户请求
    │
    ▼
┌─────────────┐
│    Nginx    │  ← 反向代理、负载均衡、静态资源缓存
└──────┬──────┘
       │
       ▼
┌─────────────┐
│   Tomcat    │  ← Spring Boot 内嵌容器 (端口7002)
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Interceptor │  ← 登录拦截器验证
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  Controller │  ← 接收请求、参数绑定
└──────┬──────┘
       │
       ▼
┌─────────────┐
│   Service   │  ← 业务逻辑处理
└──────┬──────┘
       │
       ▼
┌─────────────┐
│    DAO      │  ← 数据访问
└──────┬──────┘
       │
       ▼
┌─────────────┐
│  MySQL/     │  ← 数据持久化
│   Redis     │
└─────────────┘
```
